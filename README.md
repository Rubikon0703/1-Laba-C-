## Задача 1

### Текст задачи

Разработать класс с одним строковым полем. Создать конструктор копирования. Разработать
метод, создающий строку из первого и последнего символов поля. Перегрузить метод ToString()
для формирования строки из полей класса. Реализовать дочерний класс (его содержание
предложить самостоятельно и описать в решении: какой содержательный смысл имеют поля;
реализовать конструкторы; предложить и реализовать 2-3 метода). Протестировать все
конструкторы и другие методы базового и дочернего классов.

### Алгоритм решения
БАЗОВЫЙ КЛАСС:
Конструктор по умолчанию: инициализирует поле пустой строкой "".

Конструктор с параметром (строка): присваивает полю переданное значение.

Конструктор копирования: принимает объект того же класса и копирует его строковое поле в текущий объект.

Свойство (геттер и сеттер) для доступа к строковому полю (если нужно).

Метод FirstLast():

Проверяет, не пуста ли строка (или не равна null).

Если пуста – вернуть пустую строку.

Иначе берет первый символ ([0]) и последний символ ([Length-1]), объединить их в новую строку и возвращает.

ToString()

ДОЧЕРНИЙ КЛАСС:

Конструктор по умолчанию: вызывает базовый конструктор, устанавливает key = 0.

Конструктор с параметрами (строка, ключ): вызывает базовый конструктор со строкой, затем шифрует эту строку (вызов отдельного метода) и сохраняет ключ.

Конструктор копирования: принимает объект дочернего класса, вызывает базовый конструктор копирования, копирует ключ.

МЕТОДЫ ДОЧЕРНЕГО:
Метод шифрования: проходит по всем символам строки, сдвигает их код на значение ключа, сохраняет результат в поле.

Метод расшифровки: возвращает расшифрованную строку (обратный сдвиг), не изменяя поле.

Метод смены пароля: принимает новую строку и новый ключ, шифрует и сохраняет.

Метод маскировки: выводит на экран строку из звёздочек длиной, равной длине зашифрованного текста.

Статический метод сравнения: принимает два объекта дочернего класса, расшифровывает их и сравнивает длины.

# Задание 3

## Задача 1

### Текст задачи

Унарные операции:
++ добавление минуты к объекту типа Time
-- вычитание минуты из объекта типа Time
Операции приведения типа:
byte (явная) – результатом является количество часов (минуты
отбрасываются);
bool (неявная) – результатом является true, если часы и минуты не равны
нулю, и falseв противном случае.
Бинарные операции:
+ Time t, беззнаковое целое число (минуты) (лево- и правосторонние
операции). Результат должен быть типа Time.
– Time t, беззнаковое целое число (минуты) (лево- и правосторонние
операции).Результат должен быть типа Time.



### Алгоритм решения
Оператор ++ – добавляет одну минуту к объекту Time.
Оператор -- – вычитает одну минуту из объекта Time.
Оператор Явное приведение к byte – возвращает количество часов (минуты отбрасываются).
Оператор Неявное приведение к bool – возвращает true, если время не равно 00:00 (часы или минуты не нулевые), иначе false.
Оператор Time + uint и uint + Time – добавляет заданное количество минут (беззнаковое целое) к объекту Time, результат – новый Time.
Оператор Time - uint и uint - Time – вычитает заданное количество минут.
